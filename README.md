## 🛠 Commands

Install packages:
```bash
npm install
```
Run the project:
```bash
npm run serve 
```

## 🔒 API limit

> Free usage assumes no more than 1000 free API calls per day.
For more information, see the [API documentation](https://openweathermap.org/api).
